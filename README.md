# 1. 概要（Overview）

**価値場理論（Value-Field Theory）** は、
個人・組織・社会の変化を、次の 4 つの変数の循環として記述する
**期待値ベースの動学的枠組み（Dynamic Expectation Framework）** である。

| 変数    | 名称                                   | 概要                                      |
| ----- | ------------------------------------ | --------------------------------------- |
| **P** | 背景構造（Background Field）               | 制度・文化・安全性・人口構造など、主体には観測されず期待値として推定される要因 |
| **K** | 自由度（Capability / Degrees of Freedom） | 選択可能性・資源・地位・機会など、主体の可動域                 |
| **X** | 選好ベクトル（Preference Vector）            | 主体が「どちらへ動こうとするか」という方向ベクトル               |
| **A** | 行為（Action）                           | 実際に観測される行動。X と反応係数 α(t) の結合で生じる         |

価値場理論は、これらの要素が
**P → K → X → A → K → P**
のように循環的に更新される構造を扱う。

---

# 2. 各変数の定義

## P：背景構造（Background Field）

* 制度・文化・安全性・人口構造・環境など
* 主体からは **直接観測されず、E[P]（期待推定）** の形で扱われる
* 変化は長期的・緩やか
* 自由度 K を規定する基盤

---

## K：自由度（Capability）

* 選択可能域、資源、安定性、立場、経済状況
* 主体は最初 **E[K]（自由度の期待値）** として捉え、経験で更新
* 中期スケールで変化
* 背景構造 P に依存

---

## X：選好ベクトル（Preference Vector）

* 「どちらへ向かおうとするか」という方向
* 時間幅 T によって意味が変わる

  * 短期：瞬間選好
  * 中期：行動傾向
  * 長期：展望・価値観
* 初期は **E[X]** として推定される

---

## A：行為（Action）

行為 A は、選好 X が反応係数 α(t) を通じて実現する：

[
A(t) = \alpha(t) X(t)
]

* **X**：方向
* **α(t)**：反応性・実行力・疲労・情動などの影響
* **A**：観測される行動

---

# 3. 期待値モデル（Expectational Form）

短期の P・K・X はいずれも不確実性を含むため
**E[P], E[K], E[X] の期待値として扱う**。

経験の蓄積により：

[
(E[K_t], E[X_t]) \rightarrow (K^*, X^*)
]

のように収束することがある。

---

# 4. 循環構造（P → K → X → A → K → P）

価値場理論の中心は、**変化が一方向ではなく循環的に更新されること**。

### 1. P → K

背景構造（E[P]）が自由度 K を規定する。

### 2. K → X

自由度 K が選好 X の方向を制約・誘導する。

### 3. X → A

選好 X が反応係数 α(t) により行為 A へ写像される。

### 4. A → K

行為 A によって資源・地位・機会が変化し、K が更新される。

### 5. K → P

更新された K が、背景構造 P に対する期待 E[P] を再評価させる。

---

# 5. 時間スケールによる見え方の違い

| 時間幅      | 中心に見える変化 | 説明             |
| -------- | -------- | -------------- |
| **短期**   | X・A      | 即時の選好方向と行為の揺らぎ |
| **一定期間** | K・X      | 行動傾向の安定化       |
| **中期**   | K        | 自由度の拡大・縮小      |
| **長期**   | P・K      | 制度・環境・社会的背景の変化 |

---

# 6. 動的安定（Dynamic Stability）

価値場理論は均衡モデルではないが、
背景構造 P の変化が緩やかな「一定期間」では

[
E[\dot{K}] \approx 0,\qquad E[\dot{X}] \approx 0
]

のような **安定状態（stable configuration）** が観察されることがある。

本質は均衡ではなく、**循環的な更新の構造**。

---

# 7. 参考にしている一般概念

* 経済学（選好・期待）
* 行動理論（反応係数・選好の揺らぎ）
* システム論（フィードバック）
* 社会科学（制度・文化の影響）

ただし、これらを引用するのではなく、
価値場理論は「P–K–X–A の関係」として再構築している。

# 8. ミクロ／マクロ／行動理論との射影（Projection Table）

**目的**：既存理論体系が価値場のどこへ写像されるか整理する。

## 8.1 総合射影表

| 理論               | 対応する価値場    | 主に説明する構造     |
| ---------------- | ---------- | ------------ |
| **ミクロ（Micro）**   | K, X, A    | 個人の自由度・選好・行為 |
| **マクロ（Macro）**   | P, K（長期）   | 制度・環境・社会構造   |
| **行動（Behavior）** | X, α(t), A | 選好の揺らぎ・反応係数  |

---

## 8.2 ミクロ理論の射影

| ミクロの概念         | 価値場 | 意味        |
| -------------- | --- | --------- |
| 予算制約・資源        | K   | 制約集合      |
| 選好（preference） | X   | 選好方向      |
| 効用最大化行動        | A   | 観測される行動   |
| 外生条件           | P   | 背景構造（期待値） |

---

## 8.3 マクロ理論の射影

| マクロの概念    | 価値場   | 意味          |
| --------- | ----- | ----------- |
| 制度・文化・環境  | P     | 背景的条件       |
| 技術・資本     | K（長期） | 長期自由度       |
| 社会的価値観    | E[X]  | 集団の方向性      |
| 経済成長・社会変化 | P→K→X | 背景→自由度→選好変化 |

---

## 8.4 行動理論の射影

| 行動の概念    | 価値場  | 意味          |
| -------- | ---- | ----------- |
| バイアス     | X    | 選好の歪み       |
| 認知負荷・情動  | α(t) | 反応性の変化      |
| ランダム選択   | A    | 行為の確率性      |
| サンクコスト効果 | K←A  | 行為→自由度更新の歪み |

---

## 8.5 射影マトリクス（1ページまとめ）

| Value-Field 変数 | Micro  | Macro | Behavior |
| -------------- | ------ | ----- | -------- |
| **P（背景構造）**    | 外生条件   | ★中核   | —        |
| **K（自由度）**     | ★制約集合  | 長期構造  | 認知的に主観化  |
| **X（選好）**      | ★選好の中心 | 集団価値観 | ★揺らぎの主役  |
| **A（行為）**      | ★行動の中心 | 集計行動  | ★反応の結果   |
| **α(t)**       | 暗黙的    | 外生    | ★中心変数    |
